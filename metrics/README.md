# Metrics

Generated outputs from the development scripts.

## Files

| File | Description | Generated By |
|------|-------------|--------------|
| `detectiondata.csv` | Detection inventory in CSV format | `toml_to_csv.py` |
| `navigator.json` | MITRE ATT&CK Navigator layer | `toml_to_navigator.py` |
| `latestdetections.md` | Markdown summary of detections | `toml_to_md.py` |

## ATT&CK Navigator

Import `navigator.json` into [MITRE ATT&CK Navigator](https://mitre-attack.github.io/attack-navigator/) to visualize detection coverage.

## Regenerating Metrics

```bash
cd development
python toml_to_csv.py
python toml_to_navigator.py
python toml_to_md.py
```
