## What is Security Operations?

Threat Intelligence
 - Processes IoC (Indicators of Compromise)

Incident Response
 - Respnd to vendor based and custom security alerts
 - provides feedback to other teams. 

Threat Hunting
 - Looks for malicious activity that is already present in any environment. 
 - Finds visibility or tooling gaps within the environment

Detection Engineering
 - Creates custom alerts for the Incident Response team.
 - Creates unit tests to confirm working detections & capabilities.
